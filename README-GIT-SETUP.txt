========================================

Your INTER-SCHOOL FEST website is ready! Here's how to push it to GitHub:

STEP 1: Install Git for Windows
---------------------------------
1. Go to: https://git-scm.com/download/win
2. Download the latest 64-bit version
3. Run the installer and follow the setup wizard
4. Choose "Git from the command line and also from 3rd-party software"
5. Select "Use Windows' default console window"
6. Complete the installation

STEP 2: Create GitHub Repository
---------------------------------
1. Go to https://github.com/sanju-babu123
2. Click the "+" icon → "New repository"
3. Repository name: evoka-2025
4. Description: INTER-SCHOOL FEST • Evoke'2k25 Website
5. DO NOT initialize with README (we have code)
6. Click "Create repository"

STEP 3: Push Your Code
-----------------------
After installation, open Command Prompt and run these commands:

1. Navigate to your project:
   cd "C:\Users\LACPUCL22\Downloads\illumina-2025-master\illumina-2025-master"

2. Initialize git:
   git init

3. Add remote origin:
   git remote add origin https://github.com/sanju-babu123/evoka-2025.git

4. Add all files:
   git add .

5. Commit changes:
   git commit -m "Initial commit: INTER-SCHOOL FEST Evoke'2k25 - Complete website with registration, admin panel, and rules"

6. Push to GitHub:
   git push -u origin master

STEP 4: If Authentication is Required
--------------------------------------
GitHub may ask for authentication. You can:

Option A: Use Personal Access Token
1. Go to GitHub.com → Settings → Developer settings → Personal access tokens
2. Generate new token with "repo" permissions
3. Use your GitHub username and the token as password

Option B: Use SSH (if you have SSH keys set up)
1. GitHub → Settings → SSH and GPG keys
2. Copy your public SSH key to GitHub
3. Use: git remote add origin git@github.com:sanju-babu123/evoka-2025.git
